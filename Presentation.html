    •  On the Boards [top]

      At launch, the App looks like this :

      By left clicking on a cell, you can place an obstacle on it.

      By right clicking on a cell with an obstacle, you can clear the cell.

      And by pressing Ctrl or Alt while left clicking, you can change the location of the starting or ending point respectively.
      It can be moved on either obstacles or available cells.

      All actions on one of the board is reflected on the other one.

      I only implemented 3 types of algorithm so far :
      The A* search algorithm, the Breadth-First search algortihm and the Dijkstra algorithm.

      A* search algorithm

      Each time a specific node is searched, the algorithm assigns a weight to its neighbouring nodes and places those in the waiting queue.
      Then, it sorts the queue from 'lightest' to 'heaviest'.
      With the lightest node being the next one to be searched.
      This process is repeated until a path is found.



      BFS search algorithm

      The Breadth-First search uses follows the First-In-First-Out principle.
      Each time a specific node is searched, its neighbours are added to the queue.
      The first node in the queue is then searched.
      This process is repeated until a path is found.



      Dijkstra algorithm

      The Dijkstra algorithm is similar to the A* search algorithm, as it also assigns weights to nodes and sorts the queue before choosing the next node.
      The difference between both algorithm is the way each node's weight is calculated.
